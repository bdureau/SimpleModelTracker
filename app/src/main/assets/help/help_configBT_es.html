Configuración del módulo Bluetooth

Para configurar su módulo Bluetooth para conectar su altímetro con su teléfono, primero deberá configurarlo para que funcione a la misma velocidad que la placa del altímetro.

La siguiente pantalla le ayudará a configurar un módulo Bluetooth para que pueda usarlo con su placa. Los módulos como HC05, HC06, etc. deberían funcionar

Para programar el módulo necesita un cable TTL

Un adaptador USB (USBC o micro USB dependiendo del dispositivo Android)

Un adaptador JST y algunos cables JST son necesarios

Deberá configurar el módulo para que funcione con su placa del altímetro.

Tenga en cuenta que estos módulos suelen venir preconfigurados para trabajar a 9600 baudios. Sin embargo, su placa del altímetro trabajará de forma predeterminada a 38400 baudios, por lo que debe asegurarse de conectarse a la velocidad correcta.

PRECAUCIÓN: asegúrese de no desconectar el módulo mientras realiza lo siguiente

Recuperar la configuración del módulo

Conecte el módulo a su dispositivo Android utilizando un cable TTL y un adaptador USB que pueda funcionar con micro USB o tipo C USB

Deberá seleccionar el módulo de Bluetooth que tiene. Si no sabe cuál tiene, seleccione el predeterminado o cualquier otro compatible. Si no puede usar ningún módulo, envíeme las referencias del módulo y/o las imágenes del módulo.

Luego, haga clic en [recuperar cfg], esto intentará conectarse al módulo utilizando la velocidad de transmisión de 38600 baudios predeterminada. Si falla, intente de nuevo, utilizará varias velocidades de transmisión hasta que se conecte. Entonces recuperará la configuración del módulo para que pueda cambiarlo

Guardar la configuración del módulo

Cambie la configuración que desee cambiar, como el nombre del módulo o la velocidad. Asegúrese de haber seleccionado una velocidad de transmisión que funcione con su tablero y haga clic en [Guardar cfg]. Esto guardará la configuración. Si ocurre algún error al guardar, se abortará el guardado

Tenga en cuenta que deberá repetir los mismos pasos para cada módulo de Bluetooth que desee configurar
