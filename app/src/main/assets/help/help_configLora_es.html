Configuración del módulo eByte Lora E220

Para realizar telemetría de largo alcance, deberá reemplazar su módulo Bluetooth por un módulo de telemetría de largo alcance, como el Lora E22O.

Los siguientes módulos se han configurado con éxito

E220-400T30D

E220-400T30S

E220-900T30D

E220-900T30S

E220-400T22D

etc ...

Para programar el módulo necesita un cable TTL

un adaptador USB (USBC o micro USB dependiendo del dispositivo Android)

se requiere un adaptador jst y algunos cables jst

Tenga en cuenta que la batería de su dispositivo Android debe ser lo suficientemente fuerte como para alimentar el módulo Lora. Considere el uso de una batería externa si tiene un módulo potente.

Para entrar en el modo de configuración, deberás poner ambos jumpers.

Deberás configurar el módulo para que funcione con tu placa altímetro.

Ten en cuenta que estos módulos suelen venir preestablecidos para trabajar a 57600 baudios. Sin embargo, tu placa altímetro trabajará de forma predeterminada a 38400 baudios, por lo que deberás asegurarte de conectarte a la velocidad correcta.

Ten en cuenta que los módulos Lora vienen en pareja, por lo que tendrás que configurar tanto el transmisor como el receptor.

PRECAUCIÓN: asegúrate de no desconectar el módulo mientras realizas lo siguiente

Recuperar la configuración del módulo

Conecta el módulo a tu dispositivo Android utilizando un cable OTG o cualquier adaptador USB que pueda funcionar con USB tipo C o micro USB.

Luego, haz clic en [recuperar cfg]. Esto intentará conectarse al módulo utilizando 9600 baudios. Luego, recuperará la configuración del módulo para que puedas cambiarla.

Guardar la configuración del módulo

Cambie la configuración que desee cambiar. Asegúrese de haber seleccionado una velocidad de transmisión que funcione con su tablero (por defecto 38400). Asegúrese de que los canales sean los mismos para el transmisor y el receptor. Si establece una clave de cifrado, asegúrese de que sea la misma en ambos. Una vez hecho esto, haga clic en [Save cfg]. Esto guardará la configuración. Si se produce algún error al guardar, se abortará el guardado

Tenga en cuenta que deberá repetir el mismo paso para cada módulo

Antes de usarlo, asegúrese de quitar ambos jumpers para salir del modo de configuración

Los módulos de telemetría Lora actualmente no funcionan con el tablero ESP32.
