Simple model tracker configuration help file

The application look and feel as well as behavious can be configured with a couple of screen

On the first screen:

- choose the color of the line on the map

- you can for the application help language, as english is the main language for the application it will always be up to date

- application units, you can choose between metrics and imperial

- baud rate can be change if you are using some slow telemetry module

- connection type can be bluetooth or USB. Bluetooth is more reliable if your USB cable is a bit loose

- Model type will allow you to select various icons depending on the model that you want to track

On the second screen:

This is for the telemetry voice. As you might want to visually look for your model it is sometime more convenient to listen to events. You can choose the events that you want to listen to as well as the voice for those
